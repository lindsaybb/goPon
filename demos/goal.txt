regOne: manually register one ONU at a time
deRegOne: manually deregister one ONU at a time

regMany: register multiple ONU at once from a file
deRegMany: deregister multiple ONU at once from a file

addSp: manually add a service profile to an onu by Serial Number
remSp: manually remove a service profile from an onu by Serial Number

//
these are a collection of individual actions that will be called up by a "stateful" initiator
in other words, once a database is established a program can easily say "look like I do"
and call these actions together with some control towards read/update/delete flow

// 
to complete the demo, collect these individual programs within a "Commander"
this single program can be called with flags to enact certain actions
